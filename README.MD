# 💰 ApexSpend - Expense Tracker

A modern and responsive Expense Tracker Web Application built using HTML, CSS, and JavaScript. ApexSpend helps users manage income and expenses, track transaction history, visualize spending patterns, and export financial data efficiently.

---

## 🚀 Features

✅ Dashboard Overview

✅ Add Income & Expense Transactions

✅ Transaction History Management

✅ Search & Filter Transactions

✅ Category-wise Expense Tracking

✅ Interactive Charts & Analytics

✅ Profile Management

✅ Dark Mode Support

✅ CSV Export Functionality

✅ Responsive User Interface

---

## 📸 Project Screenshots

### 🏠 Dashboard
Displays total balance, income, expenses, charts, and recent transactions.

### ➕ Add Transaction
Add income or expense records with category, date, and notes.

### 📜 Transaction History
View, search, filter, edit, and delete transactions.

### 👤 Profile Page
Manage user profile information and customize avatar.
